# ⚖️ نظام حمورابي - منصة إدارة المكاتب القانونية

<div dir="rtl">

## 🌟 نظرة عامة
نظام **حمورابي** هو منصة متكاملة ومتقدمة مخصصة لإدارة المكاتب القانونية والمحامين في العراق. تم تصميم النظام لتبسيط العمليات القانونية، وتنظيم ملفات الموكلين، وتحليل القضايا باستخدام الذكاء الاصطناعي بناءً على المتن القانوني العراقي.

### 🚀 التقنيات المستخدمة
*   **الواجهة الأمامية (Frontend):** React.js مع Tailwind CSS لتصميم عصري وسريع.
*   **الخلفية (Backend):** Node.js مع Express لإدارة العمليات والطلبات.
*   **قاعدة البيانات:** MongoDB (دعم كامل لـ MongoDB Atlas).
*   **الذكاء الاصطناعي:** OpenAI API (تحليل ذكي للقضايا والاستشارات).

---

## 🛠️ المتطلبات التقنية (ماذا تحتاج لتشغيل النظام؟)

لضمان عمل النظام بكفاءة، ستحتاج إلى توفير المتغيرات التالية في ملف الإعدادات (`.env`):

### 1. قاعدة البيانات (Supabase)
يجب إعداد مشروع على **Supabase** وتشغيل كود الـ SQL المرفق لإنشاء الجداول.
*   **المتغيرات:** 
    *   `SUPABASE_URL`: رابط المشروع.
    *   `SUPABASE_KEY`: مفتاح الـ API (service_role).

### 2. الذكاء الاصطناعي (Google Gemini)
تحتاج إلى مفتاح API من **Google AI Studio** لتفعيل ميزة التحليل الذكي.
*   **المتغير:** `GEMINI_API_KEY`
*   **مثال:** `AIzaSyxxxxxxxxxxxxxxxxxxxxxx`

## 🚀 خطوات التثبيت والتشغيل

### الخطوة 1: إعداد الخادم (Backend)
```bash
cd backend
npm install
# قم بإنشاء ملف .env وأضف المتغيرات المذكورة أعلاه
npm run dev
```

### الخطوة 2: إعداد الواجهة (Frontend)
```bash
cd frontend
npm install
npm start
```

## ✨ المميزات الرئيسية
*   **👥 إدارة الموكلين:** ملف شامل لكل موكل مع إمكانية رفع الصور والمستندات.
*   **📂 تنظيم الملفات:** نظام مجلدات آلي لكل موكل (قضايا، عقود، مراسلات).
*   **⚖️ المكتبة القانونية:** وصول سريع للقوانين العراقية (المدني، العقوبات، الأحوال الشخصية).
*   **🤖 التحليل الذكي:** زر "تحليل AI" لدراسة القضية وتقديم نصائح قانونية فورية.
*   **🔒 نظام الصلاحيات:** لوحة تحكم خاصة للأدمن لإدارة حسابات المحامين وتفعيلها.
