# ⚖️ نظام حمورابي - منصة إدارة المكاتب القانونية

<div dir="rtl">

## 🌟 نظرة عامة

نظام **حمورابي** هو منصة متكاملة ومتقدمة مخصصة لإدارة المكاتب القانونية والمحامين في العراق. تم تصميم النظام لتبسيط العمليات القانونية، وتنظيم ملفات الموكلين، وتحليل القضايا باستخدام الذكاء الاصطناعي بناءً على المتن القانوني العراقي.

### 🚀 التقنيات المستخدمة

| المكون | التقنية |
|-------|---------|
| **الواجهة الأمامية** | React.js مع Tailwind CSS |
| **الخلفية** | Node.js مع Express |
| **قاعدة البيانات** | Supabase (PostgreSQL) |
| **الذكاء الاصطناعي** | Google Gemini API |
| **المصادقة** | JWT (JSON Web Tokens) |
| **تشفير كلمات المرور** | bcryptjs |

## ✨ المميزات الرئيسية

### 👥 إدارة الموكلين
- ملف شامل لكل موكل مع صورة وبيانات شخصية
- إمكانية البحث والتصفية السريعة
- تحديث وحذف بيانات الموكلين

### 📂 تنظيم الملفات
- نظام مجلدات آلي لكل موكل:
  - القضايا
  - عقود الأتعاب
  - المستندات العامة
  - المراسلات
  - الأحكام والقرارات

### ⚖️ إدارة القضايا
- إنشاء وتتبع القضايا
- تحديث حالة القضايا (جارية، مغلقة، معلقة)
- إضافة جلسات المحاكمة
- تسجيل الأحكام والقرارات

### 🤖 التحليل الذكي
- زر "تحليل AI" لدراسة القضية
- تحليل ذكي بناءً على المتن القانوني العراقي
- تقديم نصائح قانونية فورية
- استخراج المواد القانونية ذات الصلة

### 📊 التقارير والإحصائيات
- لوحة تحكم شاملة
- إحصائيات الموكلين والقضايا
- رسوم بيانية للأداء
- تقارير شهرية

### 🔒 نظام الصلاحيات
- لوحة تحكم خاصة للأدمن
- إدارة حسابات المحامين
- تفعيل/تعطيل الحسابات
- إنشاء دعوات للمحامين الجدد

### 📚 المكتبة القانونية
- وصول سريع للقوانين العراقية
- تشريعات القانون المدني
- قانون العقوبات
- قانون الأحوال الشخصية

## 🛠️ المتطلبات التقنية

### البرامج المطلوبة
- **Node.js** v14 أو أحدث
- **npm** أو **yarn**

### الخدمات الخارجية المطلوبة

#### 1. Supabase (قاعدة البيانات)
- إنشاء مشروع مجاني على [supabase.com](https://supabase.com)
- الحصول على `SUPABASE_URL` و `SUPABASE_KEY`

#### 2. Google Gemini API (اختياري - للتحليل الذكي)
- الحصول على مفتاح API من [Google AI Studio](https://makersuite.google.com/app/apikey)

## 🚀 خطوات التثبيت والتشغيل

### الخطوة 1: استنساخ المستودع

```bash
git clone https://github.com/roxrj/hammurabi-law-system.git
cd hammurabi-law-system
```

### الخطوة 2: تثبيت التبعيات

```bash
npm install
```

### الخطوة 3: إعداد متغيرات البيئة

```bash
cp .env.example .env
```

ثم عدّل ملف `.env` بالمتغيرات الصحيحة:

```env
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_key
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_gemini_key
```

### الخطوة 4: إعداد قاعدة البيانات

1. اذهب إلى لوحة تحكم Supabase
2. اذهب إلى **SQL Editor**
3. انسخ محتوى ملف `database/schema.sql`
4. الصق الكود وشغّله

### الخطوة 5: تشغيل المشروع

```bash
# تشغيل Frontend و Backend معاً
npm run dev

# أو تشغيل كل واحد بشكل منفصل
npm start          # Frontend فقط
npm run server     # Backend فقط
```

## 📖 دليل الاستخدام

### للمحامين:
1. **تسجيل الدخول** باستخدام بيانات الدخول المرسلة من الأدمن
2. **إضافة موكل جديد** من قائمة الموكلين
3. **إنشاء قضايا** لكل موكل
4. **رفع المستندات** المتعلقة بالقضايا
5. **استخدام التحليل الذكي** للحصول على نصائح قانونية

### للأدمن:
1. **إدارة المحامين** - إضافة وتفعيل/تعطيل الحسابات
2. **عرض الإحصائيات** - متابعة أداء المكتب
3. **إدارة النظام** - ضمان سلامة البيانات
